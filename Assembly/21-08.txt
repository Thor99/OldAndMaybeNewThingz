- Unsigned diz que aquela varável não guarda sinal.

- bit: 0 se for positivo, 1 se negativo

- complemento a 2: Além do bit de sinal, os outros bits vão estar invertidos. Inverte bits e soma 1. Vantagem: Fazer contas como se fosse soma de números positivos.

exemplo: 3 - 5

0 0 1 1        : 3
0 1 0 1        : 5
